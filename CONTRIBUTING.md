# 贡献指南

感谢你愿意让这个宠物项目变得更好。

## 提交建议

- Issue：用于反馈显示异常、安装问题、文档问题或新的想法。
- Pull Request：用于提交明确的修复或文档改进。

## 文件要求

- 不要提交本地缓存、临时文件或编辑器配置。
- 宠物资源请放在 `pet/` 目录。
- 如果修改 `pet.json`，请确认 JSON 格式有效。
- 如果替换 `spritesheet.webp`，请确认图集仍符合 Codex 宠物规格。

## 提交信息

建议使用简短清晰的提交信息，例如：

```text
docs: update installation guide
fix: correct pet metadata
assets: update spritesheet
```
