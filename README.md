# Codex Pet

一个准备开源的 Codex 桌面宠物项目。

> 当前仓库是发布模板：请把你的宠物文件放进 `pet/` 目录后再推送到 GitHub。

## 文件结构

```text
.
├── pet/
│   ├── pet.json
│   └── spritesheet.webp
├── README.md
├── LICENSE
├── CONTRIBUTING.md
└── .gitignore
```

## 使用方式

1. 下载或克隆本仓库。
2. 将 `pet/pet.json` 和 `pet/spritesheet.webp` 放到 Codex 支持的宠物目录。
3. 在 Codex 中启用这个宠物。

## 宠物文件

Codex 宠物通常包含：

- `pet.json`：宠物名称、描述、动画配置等元数据。
- `spritesheet.webp`：8 x 9 的透明背景动画图集。
