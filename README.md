# Mika

一个可爱的 Codex 桌面宠物。

Mika 是一个长发 JK 风格的 chibi 数字宠物，穿着灰色校服、白色 T 恤、蓝白领带、格子裙、围巾和耳机。她会生气、认真思考、偷懒打盹，也喜欢小甜点。

## 文件结构

```text

```

## 使用方式

1. 下载或克隆本仓库。
2. 将 `pet/pet.json` 和 `pet/spritesheet.webp` 放到 Codex 支持的宠物目录。
3. 在 Codex 中启用这个宠物。

## 宠物文件

Codex 宠物通常包含：

- `pet.json`：宠物名称、描述、动画配置等元数据。
- `spritesheet.webp`：8 x 9 的透明背景动画图集。
